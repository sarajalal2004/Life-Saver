# User Stories

## 👥 Users
- Admin
- Manager
- Firefighter

---

## 🔐 Authentication

- As a user, I want to register so that I can access the system.
- As a user, I want to verify my email so that my account is activated.
- As a user, I want to log in so that I can access protected features.
- As a user, I want to reset my password so that I can recover my account.
- As a user, I want to update my profile so that my information stays current.

---

## 🧑‍💼 Admin

- As an admin, I want to manage users so that access is controlled.
- As an admin, I want to assign roles so that permissions are enforced.
- As an admin, I want to activate/inactivate users so that only valid users can log in.
- As an admin, I want to manage branches and types so that system data is accurate.

---

## 🧑‍🚒 Manager

- As a manager, I want to create fire cases so that incidents are recorded.
- As a manager, I want to update cases so that information stays accurate.
- As a manager, I want to manage transports so that resources are available.
- As a manager, I want to assign and track case history so that actions are recorded.
- As a manager, I want to manage contact information so that communication is possible.

---

## 🚒 Firefighter

- As a firefighter, I want to see prioritized cases so that I respond to the most urgent ones first.
- As a firefighter, I want to view case details so that I can prepare before responding.
- As a firefighter, I want to start a case so that the system shows it's being handled.
- As a firefighter, I want to complete a case so that it is marked as finished.

---

## ⚙️ System

- The system prioritizes fire cases based on severity and conditions.
- The system suggests the nearest branch to reduce response time.
- The system tracks history of each case.
